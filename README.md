# Sample OAuth 2.0 Express server

Sample OAuth 2.0 Express server to test connection and endpoints.

## Installation

```bash
yarn
```

## Usage

```bash
node index.js
```
Install [Insomnia](insomnia.rest) or [Postman](postman.com)\
Sample endpoints are available in `./endpoints`
